# Pembuatan Website

```
hannynuraini.github.io merupakan website yang dibuat dengan menggunakan **_jekyll_** yang berisi ~~sekilas~~ gambaran tentang program .NET Bootcamp.
```

Proses pembuatan website ini adalah mengedit dan menambahkan:
* CSS
* HTML
* Markdown
